# The Origin — Game Design Document

Solo + multiplayer survival horror for Roblox. Five-level campaign where the player traces the source of something that escaped containment. Each level introduces a new entity with unique AI behavior. No weapons — only hide, run, or outsmart.

## Quick Reference

- **Genre**: Survival horror (solo + co-op 2-4 players)
- **Core loop**: Enter level → scavenge → survive threats → reach extraction → unlock next level
- **Tone**: Genuinely terrifying
- **Session length**: 10-25 minutes per run
- **Replayability**: Progression/unlocks, adaptive AI, randomized loot/spawns
- **Levels**: 5 campaign levels + hub
- **Entities**: The Patient, The Crawlers, The Mimic, The Watchers, The Origin
